# penguinc
@penguin You’ll find a modding script here that lets you modify game files. This file/script is only for learning purposes and does not damage or harm the game files.



# 1. COMMANDS   [ TERMUX INSTALLATION ]



git clone https://github.com/https-penguin/penguinc.git
cd penguinc
python -m venv venv
source venv/bin/activate
pip install colorama termcolor
python main.py
